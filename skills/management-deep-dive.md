# 경영진 심층 리서치: 주식을 산다는 것은 사람을 사는 것

$ARGUMENTS 에 대해 경영진 심층 리서치를 수행한다.

**지원하는 입력 형식**: `회사명` 또는 `인명 회사명`, 예: `메이투안`, `왕싱 메이투안`, `젠슨 황 엔비디아`

> "주식을 산다는 것은 사람을 사는 것이다. 믿을 수 있는 사람을 찾아서 장기 보유하라." —— 돤융핑
>
> "경영진을 평가하려면 아무도 보지 않을 때 그들이 무엇을 하는지를 봐야 한다." —— 버핏

## 설계 취지

대부분의 투자 분석은 경영진 평가를 표면적인 수준에 머무른다: 이력, 지분 비율, 보수. 하지만 버핏은 **경영진과 함께 식사하며 대화하는** 데 많은 시간을 쓰고, 리루는 **자신의 투자 본질은 사람에 대한 투자**라고 말하며, 돤융핑은 **주식을 산다는 것은 사람을 사는 것**이라고 말한다.

이 Skill은 `/investment-research` 5단계 경영진 평가의 **심화판**이다. 표준 투자 리서치에서 경영진 점수가 불확실(★★★ 이하)하거나, 경영진이 핵심 투자 논리일 때 이 Skill로 심층 리서치를 수행한다.

AI는 경영진과 식사할 수 없지만, 공개 정보를 통해 다음을 할 수 있다:
- **경영진의 말과 행동이 일치하는지 추적**(약속 vs 이행)
- **모든 중대 자본배분 결정의 성과 분석**
- **어려운 시기의 의사결정에서 품격을 추론**
- **직원/파트너/고객의 피드백을 통한 측면 검증**

## 실행 절차

### 1단계: 핵심 경영진 파악 및 병렬 데이터 수집 시작

WebSearch로 다음 핵심 인물을 확인한다:

| 역할 | 이름 | 재임 기간 | 배경 | 지분/스톡옵션 |
|------|------|------|------|----------|
| CEO/회장 | | | | |
| CFO | | | | |
| 창업자(재임하지 않는 경우) | | | | |
| 실질적 지배자(CEO와 다른 경우) | | | | |
| 기타 핵심 임원 | | | | |

**주의**: "누가 의사결정을 하는가"와 "누구의 이름이 직함에 있는가"를 구분한다. 일부 기업은 창업자가 물러났어도 여전히 정신적 지주인 경우가 있다(예: 핀둬둬에서의 황정).

핵심 인물을 확인한 뒤, Task 도구로 여러 백그라운드 에이전트를 **병렬** 실행하여 다음 데이터를 수집한다:
1. 에이전트 1: CEO의 공개 발언 및 예측 기록(주주서한, 실적발표회, 인터뷰, SNS)
2. 에이전트 2: 자본배분 결정 기록(M&A, 자사주 매입, 배당, 신사업 투자)
3. 에이전트 3: 지배구조와 보수(지분 구조, 특수관계자 거래, 임원 보수)
4. 에이전트 4: 측면 검증 정보(직원 평가, 고객 피드백, 업계 평판)

### 2단계: CEO 능력범위 평가

#### 2.1 전략적 안목

CEO의 지난 5년간 공개 발언(주주서한, 실적발표회, 인터뷰, SNS)을 검색하여, 다음 질문에 대한 판단을 추출한다:

| 시점 | CEO의 판단/예측 | 실제 결과 | 정확도 |
|------|--------------|---------|:------:|
| | "우리는 X 시장이 …라고 본다" | X 시장의 실제는… | ✅/❌ |
| | "향후 3년 우리의 중점은…" | 실제 실행은… | ✅/❌ |

**핵심 질문**:
- CEO가 시장보다 앞선 올바른 판단을 한 적이 있는가?
- 모두가 낙관할 때 냉정함을 유지한 적이 있는가?
- 산업 트렌드에 대한 이해가 시장을 추종하는가, 독자적인 사고를 하는가?

#### 2.2 실행력

| 차원 | 평가 | 근거 |
|------|------|------|
| 전략에서 실행으로 | 말한 것을 실제로 해냈는가? | |
| 조직 능력 | 인재를 유치하고 유지할 수 있는가? | |
| 위기 대응 | 어려움에 처했을 때 어떻게 대처하는가? | |
| 반복 개선 속도 | 실수 후 수정하는 속도가 빠른가? | |

### 3단계: 정직성 평가(가장 중요)

**버핏**: "우리는 세 가지 자질을 찾는다: 정직함, 지혜, 그리고 에너지. 첫 번째가 없으면 나머지 둘은 오히려 해가 된다."

#### 3.1 약속 대 이행 추적

지난 3년간의 실적발표회, 주주서한, 공개 인터뷰에서 경영진이 한 **구체적인 약속**을 추출한다:

| # | 시점 | 약속 내용 | 약속한 자리 | 이행 여부 | 평가 |
|---|------|---------|---------|---------|------|
| 1 | | "2025년에 X 사업을 흑자 전환하겠다" | 2024년 실적발표회 | | ✅/⚠️/❌ |
| 2 | | "$X억을 자사주 매입하겠다" | 2024년 주주서한 | | ✅/⚠️/❌ |

**이행률 통계**:

| 약속 이행률 | 평가 |
|:---------:|------|
| >80% | 우수——말한 대로 실행 |
| 60-80% | 양호——방향은 맞지만 실행에 편차 |
| 40-60% | 우려——약속은 많지만 이행이 부족 |
| <40% | 심각한 문제——신뢰할 수 없음 |

#### 3.2 어려운 시기의 대응

회사 역사상 겪었던 중대 위기/난관(주가 급락, 실적 미스, 규제 충격, 경쟁 심화)을 검색하여, 경영진의 대응을 분석한다:

| 위기 사건 | 시점 | 경영진 대응 | 사후 평가 |
|---------|------|-----------|-------------|

**주목할 점**:
- 능동적으로 소통했는가, 회피했는가?
- 내부 원인을 인정했는가, 외부 탓을 했는가?
- 이 기회를 이용해 어렵지만 옳은 일을 했는가, 아니면 단기적으로 시장을 달래는 선택을 했는가?

#### 3.3 이해관계자에 대한 태도

| 이해관계자 | 경영진의 태도 | 근거 | 평가 |
|-----------|-----------|------|------|
| 주주 | 존중/무시/이용 | | |
| 직원 | 잘 대우함/착취/무관심 | | |
| 고객/사용자 | 고객 중심/단기적 착취 | | |
| 협력업체/공급사 | 공정한 협력/극단적 단가 압박 | | |
| 규제기관/사회 | 순응 및 협력/편법 시도 | | |

**리루**: "이해관계자에 대한 태도가 기업의 장기 생명력을 결정한다. 단기적 착취는 효율을 높일 수 있지만, 장기적으로는 생태계를 해친다."

### 4단계: 자본배분 능력

이것은 버핏이 가장 중시하는 경영진의 능력이다——**1달러를 벌 때마다, 경영진은 그것을 얼마로 불릴 수 있는가?**

#### 4.1 자본배분 결정 기록

회사의 지난 5년간 중대 자본배분 결정을 검색하여, 건별로 평가한다:

**M&A 기록**:

| 시점 | 인수 대상 | 금액 | 전략적 논리 | 사후 성과 | 점수(1-5) |
|------|---------|------|---------|---------|:---------:|

**자사주 매입 기록**:

`tools/financial_rigor.py verify-valuation`으로 매입 당시와 현재의 PER 등 밸류에이션 지표를 검증한다.

| 시점 | 매입 금액 | 평균 매입가 | 당시 PER | 사후 평가 | 점수(1-5) |
|------|---------|-----------|:------:|---------|:---------:|

**배당 기록**:

| 연도 | 배당 금액 | 배당성향 | 동기간 FCF | 지속가능 여부 |
|------|---------|:------:|---------|:---------:|

**신사업 투자**:

| 시점 | 투자 분야 | 누적 투입액 | 현재 상태 | 성과 평가 | 점수(1-5) |
|------|---------|---------|---------|---------|:---------:|

#### 4.2 자본배분 점수

| 차원 | 점수(1-5) | 설명 |
|------|:---------:|------|
| M&A 규율 | | 합리적인 가격에 인수했는가? 인수 후 통합은 어떠했는가? |
| 자사주 매입 타이밍 | | 저평가일 때 매입하고 고평가일 때 중단했는가? |
| 배당의 합리성 | | 배당성향이 FCF와 부합하는가? |
| 신사업 투자 | | 성공률은 어떠한가? 손절 규율은 어떠한가? |
| 현금 관리 | | 현금 보유고가 합리적인가? 과도하게 쌓아두고 있지는 않은가? |
| **종합 점수** | | |

**버핏의 기준**: 이상적인 경영진은 좋은 기회가 있을 때 과감히 투자하고, 좋은 기회가 없을 때는 적극적으로 자사주 매입/배당을 하며, 절대 비싼 가격에 M&A를 하지 않는다.

| 질문 | 답변 |

### 8단계: 경영진 평가 보고서 출력

#### 보고서 구조

```
1. 핵심 인물 요약(표)

3. 능력 평가
```

|------|:----:|:---------:|:----:|
| 정직성 | 35% | | |

#### 돤융핑의 "사람을 사는" 기준

> 다음 세 가지 질문에 답하라:
> 1. **이 사람은 정직한가?**(정직하고, 주주의 이익을 침해하지 않는가)
> 2. **이 사람은 능력이 있는가?**(전략적 안목+실행력+자본배분)
> 3. **당신은 이 사람에게 10년간 돈을 맡길 수 있는가?**
>
> 셋 다 "그렇다" = ★★★★★(5점)
> 앞의 둘이 "그렇다" = ★★★★(4점)
> 첫 번째만 "그렇다" = ★★★(3점)
> 첫 번째가 "그렇다"가 아니면 = ★(1점, 투자하지 않음)

## 핵심 원칙

- **정직성은 거부권 항목이다** — 능력 부족은 배울 수 있지만, 품격에 문제가 있으면 고칠 수 없다
- **말이 아니라 행동을 보라** — 경영진이 무엇을 말하는지는 중요하지 않고, 무엇을 했는지가 중요하다
- **어려움 속에서 진실을 보라** — 순풍일 때는 누구나 좋은 CEO다, 역풍일 때 진짜 실력이 드러난다
- **자본배분은 최종 시험이다** — 돈을 버는 것은 쉽지만, 번 돈을 잘 배분하는 것은 어렵다
- **경영진과 사랑에 빠지지 말라** — 객관성을 유지하라, 아무리 존경하는 사람이라도 큰 실수를 할 수 있다
