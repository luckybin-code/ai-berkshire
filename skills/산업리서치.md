# 산업 투자 리서치: 산업 사슬 전체 스캔 + 4대 거장 개별 종목 분석 프레임워크

$ARGUMENTS 산업에 대해 체계적인 산업 사슬 투자 리서치를 수행한다.

## 리서치 목표

하나의 투자 테마/논리 사슬에서 출발하여 다음을 완성한다:
1. 투자 논리 사슬의 모든 단계를 검증
2. 완전한 산업 사슬 전체 그림 작성
3. 전 세계 모든 상장 기업 스캔 (A주/홍콩주/미국주/국제)
4. 각 세부 구간의 대표 기업에 대해 4대 거장 프레임워크 분석 실행
5. 산업 레벨 투자 포트폴리오 구성 제안 출력

---

## 1단계: 투자 논리 사슬 구축 및 검증

### 1.1 논리 사슬 그리기
화살표 사슬로 "근본 트렌드"에서 "수혜 종목"까지의 인과관계를 표현한다. 예:
```
근본 트렌드 A
    → 수요 B를 유발
        → 병목/필수 수요 C를 창출
            → 수혜 산업 사슬 D
```

### 1.2 단계별 검증
논리 사슬의 각 화살표에 의문을 제기하고 근거를 찾는다:

| 단계 | 핵심 가정 | 검증 방법 | 데이터 출처 |
|------|---------|---------|---------|
| A→B | | 산업 데이터/전망 검색 | |
| B→C | | 수급 분석 검색 | |
| C→D | | 실제 사례/계약 검색 | |

### 1.3 "이미 발생한 검증 사건" 찾기
해당 논리 사슬을 뒷받침하는 **이미 계약/실현된 실제 비즈니스 사건**을 나열한다 (전망이 아님). 예: 대기업의 조달 계약, 정책 문서, 산업 보고서 등.

---

## 2단계: 산업 사슬 전체 그림 작성

### 2.1 산업 사슬 구조 그리기
산업을 상류→중류→하류→보조 단계로 분해한다. 예:
```
상류: 원자재/자원 채굴 → 소재 가공/정제
중류: 핵심 설비 제조 → 시스템 통합/엔지니어링 건설 → 신기술 연구개발
하류: 운영/서비스 → 최종 고객
보조: 검사/인증 → 유지보수 서비스 → 금융 도구 (ETF/신탁)
```

### 2.2 각 구간의 "비즈니스 특성" 파악
각 구간에 다음을 표기한다:

| 구간 | 비즈니스 모델 | 매출총이익률 구간 | 경쟁 구도 | 장벽 유형 | 경기순환성 |
|------|---------|-----------|---------|---------|--------|
| | 자원 판매/설비 판매/서비스 판매/임대 수취 | | 독점/과점/완전경쟁 | 자원/라이선스/기술/규모 | 강/중/약 |

### 2.3 "병목 구간" 표시
산업 사슬에서 공급이 가장 타이트하고, 대체가 가장 어렵고, 수익률이 가장 높은 구간을 식별한다 — 이런 구간이 대체로 최적의 투자 대상이다.

---

## AI 리서치 편향 자각: 산업 리서치의 특수한 함정

산업 리서치에서는 AI 데이터 편향이 독특한 방식으로 증폭된다:

**산업 레벨 편향**:
| 편향 유형 | 표현 양상 | 대응 방법 |
|---------|------|------|
| 성숙 산업 선호 | 전통 산업(은행/에너지/소비재)은 자료가 매우 많아 AI 분석이 "더 확실해" 보임 | 확실성은 비즈니스 모델에서 나오지, 리서치 보고서 수량에서 나오지 않는다 |
| 신흥 산업 저평가 | 신산업(AI 응용/합성생물 등)은 자료가 적어 AI 분석이 보수적으로 치우침 | "현재 데이터"가 아닌 "종착점 사고"로 산업 가치를 판단한다 |
| 대형주 선호 | 대기업 자료가 소기업보다 훨씬 많아 AI가 자연히 대형주를 추천하는 경향 | 소기업이 더 나은 위험 대비 수익을 가질 수 있으니, AI 분석 분량이 짧다고 무시하지 않는다 |
| 상장 선호 | 상장 기업만 스캔하면 산업 사슬의 핵심 비상장 플레이어를 놓치게 됨 | 반드시 비상장 기업을 검색하고 "향후 IPO 후보"로 표기한다 |
| 영어 자료 선호 | AI는 영문 자료 처리 능력이 더 강해, 중국/아시아 시장 플레이어를 저평가할 수 있음 | 반드시 중국어와 영어 정보원을 동시에 검색한다 |

**산업 사슬 스캔의 반편향 조치**:
1. 각 구간에서 "AI가 쉽게 찾는 기업"뿐 아니라, "덜 알려졌지만 우량할 수 있는 종목"도 적극적으로 검색한다
2. 정보가 희소한 소형주는 분석 분량이 짧다고 추천도를 낮추지 않는다 — 보고서 길이가 아니라 핵심 질문(비즈니스 본질, 해자, 경영진)으로 판단한다
3. 최종 보고서에 각 기업의 "정보 충분도"(A/B/C 등급)를 표기하여, 독자가 AI 분석의 신뢰도를 알 수 있게 한다

## 3단계: 전 세계 상장 기업 스캔

Task 도구를 사용해 백그라운드 Agent를 실행하여, 해당 산업의 모든 상장 기업을 전면적으로 검색한다.

### 검색 목록
- 미국주(NYSE/NASDAQ/NYSE American) 관련 기업
- A주(상해/심천) 관련 기업
- 홍콩주 관련 기업
- 기타 국제 시장(일본/한국/유럽/호주 등)
- 산업 ETF
- 주요 비상장 기업(향후 IPO 가능성)

### 각 기업에 대해 수집할 정보
- 회사명(중국어/영어)
- 종목 코드와 거래소
- 시가총액(근사치)
- 한 줄 설명(산업 사슬 내 위치와 역할)
- 순수 종목 여부(순수 원전 vs 다각화 사업 내 원전 부문)
- 소속 산업 사슬 구간

### 출력 형식
산업 사슬 구간별로 분류하여, 구간마다 표 하나에 스캔된 모든 기업을 담는다.
다시 투자 확실성 기준으로 계층화한다:
- **Tier 1**: 대형 시가총액, 순수 종목, 산업 리더
- **Tier 2**: 중형 시가총액, 순수 또는 높은 비중, 세부 분야 리더
- **Tier 3**: 소형 시가총액, 개발 단계, 고위험 고탄력성
- **Tier 4**: 다각화 기업 중 관련 사업을 보유한 대기업

---

## 4단계: 각 구간 대표 기업 4대 거장 분석

각 산업 사슬 구간의 **Tier 1과 Tier 2 기업**에 대해 다음 분석을 실행한다(Tier 3/4 기업은 간단히 평가하면 된다):

### 4.1 비즈니스 본질 (돤융핑)
- 이 기업이 산업 사슬에서 하는 일을 한 줄로 정의
- 매출 구조와 성장률
- 매출총이익률/순이익률 수준 및 추세
- 현금흐름 특성
- **돤융핑식 질문**: 이것은 좋은 사업인가? 왜인가?

### 4.2 해자 (버핏)
5대 해자 유형별 점수 (★1-5):

| 해자 | 강도 | 근거 |
|--------|------|------|
| 브랜드/가격결정력 | | |
| 전환비용 | | |
| 네트워크 효과 | | |
| 규모의 경제 | | |
| 기술/라이선스 장벽 | | |

**질문**: 10년 후에도 해자가 남아 있을까?

### 4.3 리스크 (멍거)
- 이 기업은 어떻게 실패할 가능성이 가장 높은가?
- 최악의 시나리오에서 얼마의 가치가 있는가?
- 똑똑한 사람들은 왜 사지 않는가?

### 4.4 경영진 (돤융핑+버핏)
- CEO/창업자는 누구인가? 핵심 의사결정 기록
- 지분 비율과 이해관계 정합성
- 간단 평가 (A/B/C 등급)

### 4.5 밸류에이션 스냅샷
- 현재 PE/PS/EV/EBITDA
- 동일 구간 경쟁사와 비교
- 간단 평가: 비쌈/적정/저렴

### 4.6 추천도
★1-5로 표기:
- ★★★★★ = 핵심 비중 후보
- ★★★★☆ = 위성 비중 후보
- ★★★☆☆ = 관찰 리스트
- ★★☆☆☆ = 고위험 옵션
- ★☆☆☆☆ = 비추천

## 5단계: 산업 레벨 리스크 평가 (멍거의 "체크리스트")

### 5.1 시스템 리스크 목록

| 리스크 | 확률 | 영향 | 대응 전략 |
|------|------|------|---------|
| 투자 논리 사슬의 특정 단계가 반증됨 | | | |
| 대체 기술 등장 | | | |
| 정책/규제 블랙스완 | | | |
| 수요 경기순환적 조정 | | | |
| 밸류에이션 버블 붕괴 | | | |

### 5.2 역사적 유추
역사상 유사한 산업 사슬 투자 테마를 찾아 그 최종 결말을 분석한다:
- 유추 대상 산업은 무엇인가?
- 최종 승자는 누구였나? (상류/중류/하류?)
- 대다수 투자자는 돈을 벌었나 잃었나?
- 현재 산업에 대한 시사점은 무엇인가?

### 5.3 편향 자체 점검
- 서사 편향: 스토리가 지나치게 완벽하지 않은가?
- 앵커링 효과: 최근 상승률에 앵커링되지 않았는가?
- 군중 심리: "모두가 사고 있어서" 사는 것은 아닌가?

## 6단계: 문명 트렌드 판단 (리루 프레임워크)

- 이 산업이 의존하는 근본 트렌드는 "문명 차원의 패러다임 전환"인가, "일시적 유행"인가?
- 역사상 가장 유사한 기술 혁명 유추 사례는 무엇인가?
- 10-20년 후 이 산업의 종착점은 무엇인가?
- 산업 사슬에서 어느 구간이 "승자독식"이 될 가능성이 가장 높은가?
- 어느 구간이 파괴적 혁신에 가장 취약한가?

## 7단계: 투자 포트폴리오 구성 제안

### 7.1 추천 포트폴리오
다음 구조로 출력한다:

| 계층 | 비중 비율 | 종목 | 소속 구간 | 핵심 논리 |
|------|---------|------|---------|---------|
| **핵심 비중** | 테마 비중의 50-60% | | | 가장 확실하고 해자가 가장 넓음 |
| **위성 비중** | 테마 비중의 25-35% | | | 탄력성이 크고 확실성이 다소 낮음 |
| **옵션 비중** | 테마 비중의 5-15% | | | 고위험 고수익, 전액 손실 가능 |
| **ETF 대안** | 위 전체를 대체 가능 | | | 종목 선정을 원치 않는 "간편 방안" |

### 7.2 매수/매도 신호

| 신호 유형 | 구체적 조건 |
|---------|---------|
| 비중 확대 신호 | |
| 비중 축소 신호 | |
| 전량 매도 신호 | |

### 7.3 테마 비중 상한 제안
투자 논리 사슬의 확실성과 위험도에 따라, 해당 테마가 전체 비중에서 차지할 상한 비율을 제안한다.

## 출력 요건

1. 모든 분석은 반드시 데이터 근거가 있어야 하며 출처를 첨부한다
2. Markdown 표를 사용해 핵심 데이터를 제시한다
3. 산업 사슬 전체 그림은 코드 블록의 텍스트 다이어그램으로 표현한다
4. 각 구간마다 최소 2-3개 대표 기업을 분석한다
5. 전 세계 기업 스캔은 최대한 완전하게 한다 (A주/홍콩주/미국주/국제)
6. 최종 완성 보고서는 `~/[산업명]산업사슬투자연구보고서.md`에 작성한다
7. 결론은 명확해야 하며, 구체적인 종목·비중·가격 구간 제안을 제시한다
8. 각 분석 모듈 끝에는 해당 거장의 "질문"을 붙인다

## 데이터 표본 점검 (발행 승인 프로세스)

보고서 작성 후 데이터 표본 점검을 실행하며, 통과해야만 발행할 수 있다:

```bash
# 1단계 — 표본 점검 목록 추출 (15% 무작위 표본)
python3 ~/ai-berkshire/tools/report_audit.py extract \
  --report <보고서 파일 경로>

# 2단계 — 목록의 각 항목을 신뢰할 수 있는 출처에서 취득 (skills/financial-data.md 참고)

# 3단계 — 발행 승인/반려 판정 출력
python3 ~/ai-berkshire/tools/report_audit.py verdict \
  --results '<작성 완료된 JSON>' \
  --report <보고서 파일명>
```

**【발행 승인】** 전체 통과 → 보고서 발행 가능; **【반려】** 불통과 항목 존재 → 수정 후 재심사.
