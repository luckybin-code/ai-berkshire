# 포트폴리오 관리: "기업 리서치"에서 "포트폴리오 관리"로

$ARGUMENTS 에 대해 투자 포트폴리오 점검과 최적화를 수행한다.

**지원하는 입력 형식**:
- 보유 종목 목록, 예: `텐센트30%, 메이투안20%, 마오타이20%, 엔비디아15%, 현금15%`
- 또는: `텐센트 500주 @480홍콩달러, 메이투안 1000주 @130홍콩달러, ...`
- 또는: `내 포트폴리오`(저장된 포트폴리오 파일 `reports/portfolio-latest.md`가 있는 경우)

> "분산 투자는 무지에 대한 보호막이다. 자신이 무엇을 하는지 안다면, 분산 투자는 별 의미가 없다." —— 버핏
>
> "내 평생 진짜 좋은 투자 기회는 열 손가락으로 셀 수 있을 정도였다." —— 리루

## 설계 취지

기업을 연구하는 것은 투자의 절반일 뿐이다. 나머지 절반은 **포트폴리오 차원의 의사결정**이다:
- 얼마나 살 것인가?(비중)
- 어떤 돈으로 살 것인가?(자금 출처——신규 자금인가 리밸런싱인가)
- 기존 보유 종목과 충돌하지 않는가?(상관관계)
- 최적의 포트폴리오는 어떤 모습인가?(기회비용)

버핏은 절대 한 종목을 고립적으로 보지 않는다——그는 항상 "이것이 내가 할 수 있는 최선의 선택인가?"를 생각한다.

## 실행 절차

### 1단계: 보유 종목 파싱

입력에서 현재 보유 종목을 파싱하여 다음 형식으로 표준화한다:

| 종목 | 코드 | 보유 수량 | 매입가 | 현재가 | 시가 | 비중 | 손익 |
|------|------|--------|-------|------|------|------|------|

입력에 금액 없이 비율만 있으면, 비율 기준으로 분석하면 된다.

동시에 기존 포트폴리오 파일(`reports/portfolio-latest.md`)이 있는지 확인하고, 있으면 읽어서 갱신한다.

### 2단계: 최신 데이터 확보

Task 도구로 백그라운드 에이전트를 실행하여, WebSearch로 각 보유 종목에 대해 병렬로 다음을 확보한다:
1. 현재 주가와 밸류에이션 지표(PER, PBR, 배당수익률)
2. 최근 한 분기의 핵심 재무 변화
3. 최근 주요 이벤트
4. 애널리스트 컨센서스 전망(선행 PER, 목표주가)

각 보유 종목에 대해 `tools/financial_rigor.py verify-valuation`으로 밸류에이션 데이터를 검증한다. 각 종목마다 정보 풍부도(A/B/C등급)를 표기하며, C등급 종목의 분석 결론에는 낮은 신뢰도를 표기한다.

### 3단계: 개별 종목 점검

각 보유 종목에 대해 신속한 건강 점검을 수행한다:

| 종목 | 현재 PER | 매수 논리 변화 여부 | 논문 건전도 | 비중 제안 |
|------|:------:|:--------------:|:---------:|---------|
| 텐센트 | 18x | 변화 없음 | 8/10 | 적정 |
| 메이투안 | 25x | 경쟁 심화 | 6/10 | 다소 높음, 비중 축소 검토 |

각 보유 종목에 대해 다음에 답한다:
- [ ] **오늘 이 종목을 보유하고 있지 않다면, 현재 가격에 매수하겠는가?**
- [ ] **내일부터 매매할 수 없다면, 5년간 보유해도 편안한가?**
- [ ] **매수 당시의 논리가 여전히 온전한가?**

**돤융핑**: "10년간 보유할 생각이 없는 주식이라면, 단 하루도 보유하지 말라."

### 4단계: 포트폴리오 차원 분석

#### 4.1 집중도 분석

| 지표 | 현재 값 | 권장 범위 | 판단 |
|------|-------|---------|------|
| 최대 비중 종목 | | <40% | |
| 상위 3개 종목 비중 합계 | | 50-80% | |
| 총 보유 종목 수 | | 5-15개 | |
| 현금 비중 | | 10-30%(시장 환경에 따라) | |

**리루의 기준**: 핵심 보유 종목 3-5개, 상위 3개가 80%+ 를 차지. **단, 이는 각 종목을 철저히 연구했다는 전제가 필요하다.**

**버핏의 기준**: 핵심 보유 종목은 10개를 넘지 않되, 그 외 위성 비중은 더 늘려도 된다.

#### 4.2 상관관계 점검

보유 종목 간의 잠재적 연관성을 파악한다:

| 종목A | 종목B | 연관 유형 | 리스크 |
|-------|-------|---------|------|
| 텐센트 | 콰이쇼우 | 동일하게 중국 인터넷 업종 | 규제 리스크 공명 |
| 엔비디아 | TSMC | AI 공급망 상하류 | AI 설비투자와 동조 변동 |
| 메이투안 | 핀둬둬 | 동일하게 중국 소비 업종 | 거시 소비 동조 변동 |

**점검 목록**:
- [ ] 전체 비중의 50% 이상이 동일한 테마/업종에 노출되어 있는가?
- [ ] 전체 비중의 50% 이상이 동일한 국가/통화에 노출되어 있는가?
- [ ] 미중 관계가 악화되면 포트폴리오는 얼마나 손실을 볼 것인가?
- [ ] 글로벌 경기 침체가 오면 포트폴리오는 얼마나 손실을 볼 것인가?

#### 4.3 기회비용 분석

이것은 버핏의 가장 핵심적인 사고방식이다——**모든 1달러는 가장 높은 수익률이 나는 곳에 놓여야 한다**.

모든 보유 종목을 "기대 연수익률" 기준으로 정렬한다:

| 순위 | 종목 | 현재 비중 | 기대 연수익률 | 확실성 | 기대수익률×확실성 |
|:----:|------|:-------:|:----------:|:------:|:--------------:|
| 1 | | | | | |
| 2 | | | | | |
| ... | | | | | |

기대수익률 추정 방법(`tools/financial_rigor.py three-scenario`를 사용하여 계산):
- **간이 공식**: 기대 연수익률 ≈ FCF 수익률 + 기대 성장률(주된 방법)
- **가치주 검증**: 안전마진 회귀 + 이익 성장률 + 배당수익률
- **성장주 검증**: 이익 성장률 × 적정 PER의 변화

**핵심 질문**: 순위가 가장 낮은 종목의 기대수익률이 현금(무위험이자율 ~4%)보다 높은가? 그렇지 않다면 매도하여 현금으로 전환해야 한다.

#### 4.4 스트레스 테스트

| 시나리오 | 가정 | 포트폴리오 예상 영향 | 최대 낙폭 |
|------|------|-----------|---------|
| 글로벌 침체 | 기업 이익 20-30% 감소 | | |
| 미중 갈등 격화 | 중국 관련주 50% 할인 | | |
| 금리 급등 | 10년물 국채금리 → 6% | | |
| 기술주 버블 붕괴 | 기술주 PER 40% 압축 | | |

각 시나리오에 대해 정성적+대략적 평가를 수행한다(각 보유 종목의 업종 특성과 과거 밸류에이션 변동폭 기준):
- 어떤 종목이 가장 큰 타격을 받는가? 대략적인 영향 방향과 규모 범위
- 포트폴리오 전체가 이를 감당할 수 있는가?
- 헤지가 필요한가?

### 5단계: 최적화 제안

#### 5.1 리밸런싱 제안

위 분석을 바탕으로 구체적인 리밸런싱 제안을 제시한다:

| 조치 | 종목 | 현재 비중 | 권장 비중 | 이유 |
|------|------|:-------:|:-------:|------|
| 비중 확대 | | | | |
| 비중 축소 | | | | |
| 전량 매도 | | | | |
| 신규 편입 | | | | |
| 유지 | | | | |

#### 5.2 대체 종목 찾기

포트폴리오에 "현금보다 못한" 비중이 있거나, 현금 비중이 지나치게 높다면, `/industry-research` 또는 `/investment-checklist`를 사용해 관심 있는 업종/기업을 체계적으로 스크리닝할 것을 제안한다. 본 Skill 내에서 직접 개별 종목을 추천하지 않는다.

#### 5.3 현금 관리

| 현재 현금 비중 | 권장 현금 비중 | 이유 |
|:----------:|:----------:|------|

**버핏**: 현재 3,820억 달러의 현금을 보유하고 있으며, 이는 총자산의 25% 이상을 차지한다——좋은 기회를 찾지 못했을 때는 현금이 최선의 포지션이다.

### 6단계: 포트폴리오 보고서 출력

#### 보고서 구조

```
1. 포트폴리오 개요(보유 종목표+비중 구성 설명)
2. 개별 종목 점검(각 종목의 건강 상태)
3. 포트폴리오 분석
   - 집중도: 과도하게 분산/집중되어 있는가?
   - 상관관계: 잠재적 연관성과 리스크 공명
   - 기회비용: 순위가 가장 낮은 종목이 보유할 가치가 있는가?
   - 스트레스 테스트: 극단적 시나리오에서의 낙폭 예상
4. 리밸런싱 제안(구체적 조치+이유)
5. 다음 점검 시점과 주목할 포인트
```

#### 결론은 반드시 명확히 답해야 함

1. **포트폴리오 전체 건강도**: 우수 / 양호 / 조정 필요 / 심각한 문제
2. **지금 가장 먼저 해야 할 한 가지는 무엇인가?**(X 비중 확대 / Y 비중 축소 / 유지)
3. **현재 가장 큰 리스크는 무엇인가?**

### 7단계: 포트폴리오 파일 저장

포트폴리오 정보를 `reports/portfolio-latest.md`에 저장하며, 다음을 포함한다:
- 최신 보유 종목표
- 이번 점검 날짜와 결론
- 리밸런싱 기록(누적)
- 다음 점검 알림

---

## 핵심 원칙

- **모든 1달러에는 기회비용이 있다** — 평범한 종목을 보유하는 비용은 우수한 종목을 놓치는 것이다
- **집중은 리스크가 아니다, 무지야말로 리스크다** — 깊이 이해하는 종목 3개를 보유하는 것이, 어설프게 아는 종목 30개를 보유하는 것보다 안전하다
- **현금도 하나의 포지션이다** — 좋은 기회를 찾지 못했을 때 현금을 보유하는 것은 부끄러운 일이 아니다
- **포트폴리오 차원 > 개별 종목 차원** — 좋은 종목도 잘못된 비중에 있으면 발목을 잡을 수 있다
- **정기적으로 점검하되, 과도하게 매매하지 말라** — 분기마다 한 번씩 점검하는 것으로 충분하며, 매일 시세를 보며 리밸런싱할 필요는 없다
