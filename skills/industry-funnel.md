# 산업 퍼널 스크리닝: 전체 시장에서 3개 종목까지 좁히는 가치투자 정선 프로세스

$ARGUMENTS 산업/방향에 대해 퍼널(깔때기) 방식의 가치투자 스크리닝을 실행하여, 전체 시장 스캔부터 단계별로 좁혀 최종 3개 후보 종목까지 정선한다.

## 적용 시나리오

산업이나 투자 방향(예: "AI 컴퓨팅", "혁신 신약", "로봇")을 제시하며 다음을 원할 때:
1. 중요한 종목을 하나도 놓치지 않기 (중국 A주, 홍콩주, 미국주, 비상장 후보 포함)
2. 통일된 기준으로 "스토리 종목"과 질이 낮은 기업을 걸러내기
3. 진짜 깊이 연구할 가치가 있는 상위 3개 종목에 역량을 집중하기
4. 각 단계마다 명확한 유지/탈락 기준을 두어 복기·추적이 가능하게 하기

`industry-research`와의 차이:
- `industry-research`는 산업 사슬 구조와 전체 그림, 세부 구간 분석에 중점
- `industry-funnel`은 개별 종목 스크리닝 퍼널에 중점을 두고, 전체 시장에서 단계별로 3개 종목까지 정선

두 스킬은 상호 보완적이다: 먼저 `industry-research`로 산업 사슬 구도를 파악한 뒤, `industry-funnel`로 종목을 정선한다.

---

## 퍼널 구조 개요

```
1단계: 전체 시장 스캔       30-60개 종목  (활성도+상승률+시가총액 상위 30의 합집합)
        ↓ 가치투자 5대 하드 지표
2단계: 1차 스크리닝          ≤ 10개 종목  (5개 지표 모두 통과 + 해자 ★★★ 이상)
        ↓ 정밀 분석
3단계: 정밀 분석             ≤ 10개 종목  (종목별 300-500자 구조화 분석)
        ↓ 최종 선정
4단계: 4대 거장 심층 분석      3개 종목    (종목별 800-1200자, 버핏·멍거·돤융핑·리루 관점)
        ↓
출력: 투자 제안 + 매매 신호 + 비중 제안
```

각 단계에서 "탈락한 종목"은 반드시 탈락 사유를 남겨야 하며, 블랙박스 처리해서는 안 된다.

## 1단계: 전체 시장 스캔 입구

### 1.1 활성 종목 정의 (3개 범주의 합집합)

**A범주 - 거래 활성도**:
- 최근 30일 일평균 거래대금이 산업 내 상위권 (A주/홍콩주/미국주 각각 상위 30개)

**B범주 - 상승률 상위**:
- 최근 30일 상승률 상위 20
- 최근 90일 상승률 상위 20
- 두 목록의 합집합

**C범주 - 시가총액 앵커**:
- 산업 내 시가총액 상위 30 (등락 무관)

최종 스캔 풀 = A ∪ B ∪ C, 예상 30-60개.

### 1.2 반드시 검색해야 할 시장

| 시장 | 출처 권장 |
|------|---------|
| A주 (상해/심천) | 통화순(同花顺)/동방재부 산업 섹터, 통다신(通达信) |
| 홍콩주 | 푸투(富途)/통화순 홍콩주, HKEX 산업 분류 |
| 미국주 | NASDAQ/NYSE 산업 ETF 편입 종목, Yahoo Finance |
| 국제 시장 | 일본·한국·대만·유럽의 관련 기업을 놓치면 안 됨 (특히 반도체, 전자) |
| 비상장 기업 | "향후 IPO 후보"를 별도 절로 분리하고 최신 밸류에이션과 잠재 IPO 시점을 표기 |

### 1.3 출력 형식

| 회사명 | 코드 | 시장 | 시가총액 | 주력 사업 한 줄 설명 | 해당 산업 매출 비중 | 선정 범주(A/B/C) |
|-------|------|-----|------|----------|-----------|----------------|

**핵심 자체 점검**:
- 산업 매출 비중 < 30%인 "곁다리 종목"은 신중히 다루고 "순수 종목 아님"으로 표기
- 중국/아시아 시장이 영문 자료가 적다는 이유로 누락되지 않도록 주의
- 소형주가 AI가 선호하는 대형주에 밀려 누락되지 않도록 주의

## 2단계: 가치투자 5대 하드 지표 1차 스크리닝 → ≤ 10개 종목

1단계에서 나온 30-60개 기업에 대해 5대 하드 지표를 종목별로 적용한다.

### 2.1 5대 하드 지표

| # | 지표 | 통과 기준 | 완화 조건 | 데이터 출처 |
|---|------|---------|---------|---------|
| 1 | PE 밸류에이션 | 합리적 (역사적 구간, 동종업계 비교) | 고성장 기업은 PEG < 1.5까지 완화 가능 | 재무제표+wind/통화순 |
| 2 | ROE | > 15% 또는 최근 3년 추세 개선 | 중자산 산업은 완화 가능 | 재무제표 |
| 3 | 영업현금흐름 | 양(+)이며 순이익의 70% 이상 | — | 재무제표 |
| 4 | 부채비율 | < 60% | 공공사업/전력은 70%까지 완화 가능 | 재무제표 |
| 5 | 해자 약식 평가 | ★★★ 이상 | — | 정성적 판단 |

**해자 5대 유형**:
- 브랜드/가격결정력
- 전환비용/사용자 락인
- 네트워크 효과
- 규모의 경제
- 기술/라이선스/자원 장벽

### 2.2 출력 형식

| 회사 | PE | ROE | 현금흐름/순이익 | 부채비율 | 해자 | 종합 | 유지/탈락 | 탈락 사유 |
|------|----|----|-----------|-------|-------|------|------|---------|

**유지 규칙**:
- 5개 지표 모두 통과 → 바로 유지
- 4개 통과 + 1개 근접 → 유지하되 노란색 표시
- 4개 미만 통과 → 탈락, 사유 명시

**목표**: ≤ 10개 종목 유지. 유지 종목이 너무 많으면(> 12개), 해자 기준을 ★★★★로 높여 재스크리닝한다.

## 3단계: 정밀 분석 (≤ 10개 종목, 종목별 300-500자)

1차 스크리닝에서 유지된 기업을 종목별로 구조화 분석한다.

### 3.1 종목별 분석 템플릿

```
## {회사명} ({코드})

**한 줄 비즈니스 모델**:
(무엇을 팔고, 누구에게 팔고, 어떻게 돈을 버는가)

**재무 품질**:
- 매출 성장률 / 이익 성장률 / 매출총이익률 / ROE / 현금흐름
- 핵심 변화 (최근 1-2년 가장 중요한 재무적 전환점)

**해자 깊이**:
- 주요 해자 유형 + 구체적 근거
- 5년 후에도 해자가 유지될지에 대한 간단한 판단

**주요 리스크 (상위 3개)**:
1.
2.
3.

**밸류에이션 약식 평가**:
- 현재 PE/PS/EV/EBITDA + 역사적 구간 내 위치
- 동종업계 비교
- 한 줄 결론: 비쌈 / 적정 / 저렴

**최종 3개 종목에 진입?**: 예 / 아니오 (사유)
```

### 3.2 최종 3개 종목 선정 기준

점수 순위로 상위 3개를 뽑는 것이 아니라, "포트폴리오 상호보완성"을 기준으로 선정한다:
- 최소 1개는 "고확실성 저탄력성" (버핏형)
- 최소 1개는 "중간 확실성 중간 탄력성" (성장형)
- 선택적으로 1개 "고탄력성 고위험" (옵션형)

특정 세부 트랙에서 충분히 좋은 3개를 찾지 못하면, 억지로 채우지 말고 "최종 2개 + 관찰 1개"로 작성한다.

## 4단계: 4대 거장 심층 분석 (3개 종목, 종목별 800-1200자)

최종 선정된 3개 종목에 대해 4대 거장 관점 심층 분석을 실행한다.

### 4.1 돤융핑 관점: 비즈니스의 본질

- 이 회사가 하는 사업을 한 줄로 정의
- 이것은 좋은 사업인가? 왜인가?
- 이 회사의 "본분"은 무엇인가? 경영진이 이탈하지는 않았는가?
- 비즈니스 모델의 "지속가능성"은 어디에 있는가?

### 4.2 버핏 관점: 해자의 깊이

- 5대 해자 유형별 점수 매기기 (★1-5), 구체적 근거 나열
- 10년 후에도 해자가 남아 있을까?
- 지금 매수할 때 "안전마진"은 어디에 있는가?

| 해자 | 강도 | 구체적 근거 |
|-------|------|---------|
| 브랜드/가격결정력 | | |
| 전환비용 | | |
| 네트워크 효과 | | |
| 규모의 경제 | | |
| 기술/라이선스 장벽 | | |

### 4.3 멍거 관점: 리스크와 실패 패턴

- 이 회사는 어떻게 실패할 가능성이 가장 높은가? (상위 3개 실패 경로 나열)
- 최악의 시나리오에서 얼마의 가치가 있는가? (극단적으로 단순화한 밸류에이션)
- 똑똑한 사람들은 왜 사지 않는가? (반대 논거)
- 도덕적/컴플라이언스/경영진 리스크가 있는가?

### 4.4 리루 관점: 문명 차원의 트렌드 포지셔닝

- 이 회사가 속한 트랙은 "문명 차원의 패러다임 전환"인가, "일시적 유행"인가?
- 역사상 가장 유사한 기술 혁명 유추 사례는?
- 10-20년 후 이 회사의 종착점은?
- 승자독식 구도인가?

### 4.5 종합 추천도

```
추천도: ★★★★☆
비중 유형: 핵심 / 위성 / 옵션 / 관찰
매수 권장 구간: 현재가 / N% 조정 시 / 인내심 있게 대기
권장 비중: 해당 테마 비중의 X%
핵심 모니터링 지표: (이 회사의 논리가 뒤집힐 때의 신호는 무엇인가)
```

## 5단계: 종합 출력

리포트 말미에 다음을 통합한다:

### 5.1 최종 선정 3개 종목 포트폴리오 표

| 회사 | 유형 | 추천도 | 권장 비중 | 핵심 논리 | 핵심 리스크 |
|------|------|-------|---------|---------|---------|
| A | 핵심 | ★★★★★ | 50-60% | | |
| B | 위성 | ★★★★☆ | 25-35% | | |
| C | 옵션 | ★★★☆☆ | 5-15% | | |

### 5.2 산업 레벨 ETF 대안

종목 선정을 원하지 않는 경우, 관련 ETF 1-3개를 (A주/홍콩주/미국주) 나열한다.

### 5.3 전체 산업 위치 판단

- 산업 PE/PB 역사적 분위
- 자금 흐름 (북향자금, ETF 신청/환매, 매도측 커버리지 밀도)
- 전체적으로 "초기/확장/성숙/쇠퇴" 중 어느 단계인지

### 5.4 정보 충분도 자체 평가 (필수)

| 차원 | 등급 | 설명 |
|-----|------|-----|
| 기업 재무 데이터 완전성 | A/B/C | |
| 밸류에이션 데이터 최신성 | A/B/C | |
| 산업 구도 판단 | A/B/C | |
| 경영진 정보 | A/B/C | |

A = 데이터 충분하고 신뢰 가능; B = 일부 결측이나 주요 결론에 영향 없음; C = 결측이 많아 결론에 신중해야 함.

### 5.5 업데이트가 필요한 데이터 항목

어떤 데이터가 추정치인지, 어떤 데이터를 추후 확인해야 하는지, 어느 분기 재무제표를 중점적으로 추적해야 하는지 명확히 나열한다.

### 5.6 자료 출처 목록

모든 데이터/결론의 출처 링크를 분류하여 나열한다 (재무제표, 리서치 보고서, 뉴스, 산업 보고서).

## AI 리서치 편향 자각 (중요)

퍼널 스크리닝 과정에서 AI가 빠지기 쉬운 함정:

| 편향 | 표현 양상 | 대응 방법 |
|------|-----|------|
| 대형주 선호 | 시가총액이 큰 기업은 자료가 많고 분석 분량이 길어져 "더 좋아 보임" | 하드 지표와 해자 점수로 판단하며, 보고서 분량으로 순위를 매기지 않는다 |
| 영어 자료 선호 | 미국주 자료가 풍부해 A주/홍콩주가 저평가되기 쉬움 | 반드시 중국어·영어 모두 검색하며, A/H 기업을 놓치지 않는다 |
| 스토리 선호 | 높은 상승률 + 언론 관심 = 더 좋아 보이는 "AI 컨셉주" | "AI 매출 비중" vs "AI 스토리 비중"을 구분하고 실제 사업을 본다 |
| 현재 실적 선호 | 현재 재무 상태가 좋은 기업이 선정되기 쉬워, 전환기의 다크호스를 놓칠 수 있음 | 2단계 1차 스크리닝에서는 "추세 개선"을 완화 조건으로 허용한다 |
| 상장 선호 | 상장 기업만 보면 트랙 내 최고의 플레이어를 놓칠 수 있음 | 반드시 "향후 IPO 후보"를 나열하고 밸류에이션과 시점을 표기한다 |

## 출력 요건

1. **리포트 위치**: `reports/{산업명}-funnel-{YYYYMMDD}.md` (산업 보고서는 reports/ 루트 디렉터리에 저장)
2. **언어**: 한국어
3. **스타일**: 직설적, 날카롭게, 불필요한 말 없이
4. **데이터**: 모든 데이터에 출처 표기; 추정치는 "추정"으로 표기
5. **선입견 배제**: 데이터를 먼저 제시 → 논리 전개 → 결론 도출
6. **양면성**: 모든 핵심 판단에 반대 논거를 첨부
7. **각 단계 탈락 기록 유지**: 탈락한 기업도 이름+사유를 남긴다

- `/management-deep-dive` —— 경영진 심층 리서치

`industry-funnel`은 입구이며, 후속 스킬은 더 깊이 파고드는 역할을 한다.
