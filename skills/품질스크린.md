# 열등주 배제 스크리닝: 7개 지표로 일류 기업이 아닌 종목을 빠르게 배제

$ARGUMENTS 에 대해 열등주 배제 지표 스크리닝을 수행하여, 일류 기업 기준에 미달하는 종목을 빠르게 배제한다.

**지원하는 입력 형식**:

| 입력 방식 | 예시 | 설명 |
|---------|------|------|
| 개별 종목 | `텐센트, 메이투안, 엔비디아` | 종목별로 스크리닝 |
| 업종 | `중국 맥주 업종` `글로벌 클라우드 컴퓨팅` `홍콩 스포츠 브랜드` | 먼저 해당 업종의 주요 상장기업(10-20개)을 검색한 뒤, 종목별로 스크리닝 |
| 시장/지수 | `항셍지수 구성종목` `CSI300` `나스닥100` | 구성종목 목록을 확보한 뒤, 종목별로 스크리닝 |
| 테마 | `중국 고배당 50선` `글로벌 AI 컴퓨팅 체인` | 먼저 테마 관련 기업을 검색한 뒤, 종목별로 스크리닝 |

업종/시장/테마 모드에서는, 출력에 통과율 통계, 업종 내 순위, 섹터 비교 요약을 추가로 포함한다.

## 설계 원칙

- **목표**: 일류 우량기업을 오판하여 배제하지 않되, 확실히 일류가 아닌 기업은 배제할 수 있어야 함
- **논리**: 7개 하드 지표 + 2개(원문 기준 3개) 면제 규칙, 놓치는 것이 있더라도 오판하여 배제하지는 말 것
- **적용 범위**: 모든 상장기업(은행/보험은 3번 이자보상배율 항목 미적용)

---

## 7개 열등주 배제 지표

| # | 지표 | 배제 조건 | 측정하는 것 |
|---|------|---------|-------------|
| 1 | 10년 평균 ROE | < 8% | 자본 효율——주주의 돈이 기회비용을 넘어서는 수익을 내는가 |
| 2 | 5년 누적 잉여현금흐름 | 마이너스 | 실질 현금——이익이 "장부상 이익"에 불과한 것은 아닌가 |
| 3 | 이자보상배율(EBIT/이자비용) | < 2배 | 상환 안전성——이자를 갚을 수 있는 능력 |
| 4 | 장기 매출총이익률 | < 15% | 가격결정력——제품/서비스에 차별성이 있는가 |
| 5 | 영업현금흐름 / 순이익(5년 평균) | < 0.7 | 이익의 질——번 이익을 실제 현금으로 회수할 수 있는가 |
| 6 | 장기 순이익률 | < 5% | 리스크 저항력——매출이 흔들릴 때 이익이 제로가 되지는 않는가 |
| 7 | 5년 총 발행주식 팽창 | > 20%(M&A 사유가 아닌 경우) | 주주 이익——경영진이 당신의 지분을 희석하고 있지는 않은가 |

## 3개 면제 규칙

### 면제규칙A: 전략적 투자기 면제(1번 지표에 적용)

다음 3가지 조건을 동시에 만족하면, 1번 ROE 미달을 면제할 수 있다:
1. 상장한 지 10년이 안 됨
2. 매출총이익률 > 30%(비즈니스 모델 자체에 가격결정력이 있음을 증명)
3. 최근 2년간 영업현금흐름이 플러스(자체 현금창출 능력이 이미 갖춰졌음을 증명)

**논리**: 높은 매출총이익률+현금흐름 흑자 전환은 비즈니스 모델에 문제가 없음을 보여주며, ROE가 낮은 것은 아직 투자기이기 때문일 뿐이다. 대표적 사례: 메이투안.

### 면제규칙B: 주동적 저이익률 면제(6번 지표에 적용)

다음 2가지 조건을 동시에 만족하면, 6번 순이익률 미달을 면제할 수 있다:
1. 매출총이익률 > 30%(벌 수 있는 능력은 있지만 벌지 않기로 선택함)
2. 최근 2년간 순이익률이 이미 5% 이상으로 회복했거나, 명확한 상승 추세를 보임

**논리**: 매출총이익률이 높다는 것은 가격결정력이 있다는 뜻이며, 순이익률이 낮은 것은 능력 부족이 아니라 전략적 선택(재투자)이다. 대표적 사례: 아마존.

### 면제규칙C: 고회전 박리다매 모델 면제(4번과 6번 지표에 적용)

다음 3가지 조건을 동시에 만족하면, 4번 매출총이익률과 6번 순이익률 미달을 면제할 수 있다:
1. ROE > 20%(이익률은 낮지만 자본수익률은 매우 높음을 증명)
2. 영업현금흐름/순이익 > 1.0(이익의 질에 문제 없음)
3. 비즈니스 모델이 "회원제/플랫폼 수수료/고회전 박리다매" 유형에 속함(이익이 상품 마진에 반영되지 않음)

**논리**: 일부 일류 기업의 이익은 매출총이익률에 숨어있지 않고, 회원비, 회전 효율, 플랫폼 수수료에 숨어있다. 이런 기업들은 매출총이익률과 순이익률이 원래 낮지만, ROE가 매우 높다는 것은 자본 효율이 일류라는 뜻이다. 대표적 사례: 코스트코(매출총이익률 12%, 순이익률 2.5%지만 ROE 25%+, 회원 갱신율 90%+).

## 실행 절차

### 1단계: 입력 파싱, 스크리닝 범위 결정

**모드 판단**:
- 입력이 구체적인 기업명/코드인 경우 → **개별 종목 모드**, 바로 2단계로 진행
- 입력이 업종/시장/테마인 경우 → **배치 모드**, 먼저 아래 작업을 수행:
  1. WebSearch로 해당 업종/시장/테마에 속한 주요 상장기업을 검색
  2. 업종 모드: 해당 업종 시가총액 상위 15-20개 상장기업 커버
  3. 지수 모드: 전체 구성종목 목록 확보
  4. 테마 모드: 관련 기업을 검색하여 15-30개 커버
  5. 전체 기업 목록을 나열하여 확인받음(기업 수가 30개를 초과하면 배치로 나누어 병렬 처리)

각 기업에 대해 정식 명칭, 코드, 거래소를 확정한다.

### 2단계: 병렬 데이터 수집

각 기업마다 독립된 백그라운드 에이전트를 실행하여, WebSearch로 다음 데이터를 검색한다:

1. **ROE**: 최근 10년(또는 상장 이후)의 연도별 ROE, 평균값 계산
2. **잉여현금흐름**: 최근 5년의 영업현금흐름과 자본적지출, 5년 누적 FCF 계산
3. **이자보상배율**: 최신 연도 EBIT와 이자비용, 배율 계산
4. **매출총이익률**: 최근 5년 매출총이익률 추세
5. **영업현금흐름/순이익**: 최근 5년의 비율, 평균값 계산
6. **순이익률**: 최근 10년 순이익률 추세, 평균값 계산
7. **총 발행주식 변화**: 5년 전과 현재의 총 발행주식, 팽창 비율 계산

데이터 출처 우선순위: 기업 연간보고서 > 증권사 리포트 > 금융 데이터 플랫폼

### 3단계: 항목별 검증

각 기업에 대해 7개 지표를 하나씩 검증한다:
- ✅ 통과
- ❌ 미통과
- ⚠️ 경계(수치와 함께 설명)

특정 지표를 위반한 경우, 해당 면제 조건을 만족하는지 확인한다.

### 4단계: 결과 출력

#### 출력 형식


## 유의사항

1. **은행/보험**: 3번 지표(이자보상배율)는 적용하지 않음, 이들의 비즈니스 모델 본질은 예대마진 경영이기 때문
2. **REIT**: ROE는 자산 재평가로 인해 변동폭이 클 수 있으므로, "핵심 영업이익 기준 ROE"로 대체
3. **데이터 부족**: 특정 데이터를 확보할 수 없으면, 바로 통과/미통과로 판정하지 말고 "데이터 부족"으로 표기
4. **경기순환 업종**: 전체 사이클(최소 고점과 저점을 각각 하나씩 포함)의 평균값을 사용하고, 단일 연도 수치는 사용하지 않음
5. **상장 기간이 짧은 경우**: 5년 미만인 기업은 확보 가능한 전체 데이터를 사용하되, 결과에 "데이터 기간 부족"으로 표기

## 한계 선언

이 지표 세트는 "확실히 좋지 않은" 기업을 배제할 수 있지만, 스크리닝을 통과했다고 해서 "확실히 좋은" 기업이라는 뜻은 아니다. 통과한 기업도 다음을 추가로 연구해야 한다:
- 비즈니스 모델의 지속가능성
- 경영진의 신뢰성
- 현재 밸류에이션의 합리성
- 경쟁 구도가 악화되고 있는지 여부

열등주 배제는 첫걸음이지, 마지막 단계가 아니다.
