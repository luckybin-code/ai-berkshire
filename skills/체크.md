# 버핏 가치투자 매수 전 체크리스트

$ARGUMENTS에 대해 버핏 가치투자 매수 전 체크리스트 분석을 실행한다.

**지원 입력 형식**: 단일 또는 다수 기업, 쉼표/모점/공백으로 구분. 예: `텐센트, 마오타이, 엔비디아` 또는 `NVDA AAPL MSFT`

## 실행 절차

### 1단계: 입력 파싱, 분석 대상 기업 전체 식별

$ARGUMENTS에서 모든 기업명/종목코드를 파싱한다. 각 기업에 대해 다음을 확정한다:
- 회사 정식 명칭, 종목 코드, 상장 거래소
- 비상장 기업일 경우 "비상장"으로 표시하고 간단한 설명을 제공(간접 투자 경로가 있는지 여부), 전체 체크리스트는 건너뛴다

### 1.5단계: AI 리서치 편향 사전 경고

각 기업에 대해 "정보 풍부도"를 신속히 등급화(A/B/C)하고, 보고서에 표기한다:

| 등급 | 판단 기준 | 체크리스트에 미치는 영향 |
|------|---------|-----------------|
| A등급 | 상장 다년, 데이터 풍부 | 정상적으로 실행하되, "합의의 함정"을 경계 — 모든 지표가 명확해 보인다고 실제로 확실한 것은 아님 |
| B등급 | 데이터가 제한적이라 추산 필요 | 각 추산 지표에 신뢰도를 표기, "좋은 사업" 판단 시 데이터 신뢰성을 가중 고려 |
| C등급 | 정보가 극도로 희소 | 6단계 표를 억지로 채우지 말고, "데이터 부족으로 판단 불가"를 정직하게 표기하며, 검증 가능한 핵심 질문에 집중 |

**핵심 원칙**: 체크리스트의 목표는 **나쁜 선택을 배제하는 것**이다. C등급 기업의 경우, "데이터 부족"은 "불통과"도 "통과"도 아니다 — AI가 표를 다 채우지 못했다고 부결 판정하지 말고, "회색 지대, 1차 정보 보강 필요"로 정직하게 표기해야 한다.

돤융핑은 이렇게 말한 바 있다: "이해가 안 된다"에는 두 가지가 있다 — 하나는 사업이 너무 복잡해서 정말 이해가 안 되는 경우, 다른 하나는 아직 시간을 들여 들여다보지 않은 경우. AI 리서치의 한계는 "자료가 적음"과 "이해가 안 됨"을 혼동하기 쉽다는 점이다.

### 2단계: 병렬 데이터 수집

Task 도구를 사용해 **각 기업마다** 독립된 백그라운드 Agent를 실행하여 데이터를 수집한다(모든 기업을 동시에 병렬로 실행), 각 Agent는 다음을 수집한다:

1. **수익성**: ROE (5-10년 추세), 매출총이익률, 순이익률, 잉여현금흐름
2. **밸류에이션 데이터**: 현재 주가, 시가총액, PE(TTM), 선행 PE, PB, 배당수익률
3. **성장 추세**: 최근 3년 매출/이익 성장률
4. **재무 건전성**: 부채 수준, 자본지출 필요, 현금 보유량, 순현금/순부채
5. **경쟁 구도**: 시장 점유율, 주요 경쟁사, 점유율 변화 추세
6. **해자 근거**: 브랜드/전환비용/네트워크효과/규모효과/기술장벽의 구체적 근거
7. **경영진 기록**: CEO 이력, 핵심 의사결정, 지분 보유, 자본배분 기록
8. **최신 동향**: 최근 6개월 주요 사건(실적, M&A, 규제, 경영진 변동 등)

### 3단계: 기업별 6관문 체크리스트 실행

이미 상장된 각 기업에 대해 순서대로 6개 관문을 통과시킨다:

---

#### 1관문: 이 사업을 이해할 수 있는가 (능력범위)

반드시 답해야 할 질문:
- [ ] 이 회사가 무엇으로 돈을 버는지 한 문장으로 말할 수 있는가?
- [ ] 10년 후에도 대체로 같은 사업을 하고 있을 것인가?
- [ ] 성패를 좌우하는 핵심 변수는 무엇인가?
- [ ] 이 산업에 대한 인식은 심층 리서치에서 온 것인가, 아니면 어디서 주워들은 것인가?

**평가 기준** (★1-5):
- ★★★★★: 비즈니스 모델이 매우 단순명료하며, 10년 확실성이 높음 (예: 마오타이 — 술을 빚어 판매)
- ★★★★☆: 모델은 명확하나 기술적 진입장벽이 있어 어느 정도 전문 지식이 필요
- ★★★☆☆: 모델은 이해 가능하나 10년 확실성이 낮고, 산업 변화가 빠름
- ★★☆☆☆: 사업 라인이 복잡하거나 산업이 격변 중이라 미래 예측이 어려움
- ★☆☆☆☆: 완전히 능력범위 밖

**하드 부결**: 돈을 버는 방식조차 설명 못 하면, 바로 "능력범위 밖, 분석 불가"로 표시한다.

---

#### 2관문: 좋은 사업인가 (경제적 특성)

데이터로 말한다, **핵심 지표는 반드시 도구로 정밀 계산해야 한다**:

```bash
python3 ~/ai-berkshire/tools/financial_rigor.py verify-valuation \
  --price {주가} --eps {EPS} --bvps {주당순자산} --fcf-per-share {주당FCF} --dividend {주당배당}
```

| 지표 | 해당 기업 수치 | 참고 기준 | 판단 |
|------|-----------|---------|------|
| ROE (5년 평균) | | >15% 우수, >20% 탁월 | |
| 매출총이익률 | | >40%는 가격결정력을 시사 | |
| 잉여현금흐름 | | 지속적으로 양(+), 순이익과 유사 | |
| 자본지출 강도 | | 경자산이 중자산보다 우수 | |
| 부채 수준 | | 이자부부채/순이익 < 3년 | |

**평가 기준** (★1-5):
- ★★★★★: ROE>25%, 고매출총이익률, 강한 FCF, 경자산, 저부채 (전부 충족)
- ★★★★☆: 4개 항목 충족
- ★★★☆☆: 3개 항목 충족
- ★★☆☆☆: 2개 항목 충족 또는 추세 악화
- ★☆☆☆☆: 대부분 미충족 또는 FCF가 지속적으로 음(-)

---

#### 3관문: 해자가 충분히 깊은가 (경쟁우위)

항목별로 점검한다:

| 해자 유형 | 보유 여부 | 구체적 근거 | 넓어지는가 좁아지는가? |
|-----------|---------|---------|--------------|
| 브랜드/가격결정력 | | | |
| 전환비용 | | | |
| 네트워크 효과 | | | |
| 원가/규모 우위 | | | |
| 기술/특허 장벽 | | | |

추가 검증: 경쟁사에게 100억을 준다면 이 사업을 복제할 수 있는가?

**평가 기준** (★1-5):
- ★★★★★: 다중 해자가 겹쳐 있고 넓어지는 중
- ★★★★☆: 최소 하나의 강한 해자가 있고 안정적
- ★★★☆☆: 해자는 있으나 충분히 깊지 않거나 추세가 불명확
- ★★☆☆☆: 해자가 잠식되고 있음
- ★☆☆☆☆: 뚜렷한 해자 없음

---

#### 4관문: 경영진을 신뢰할 수 있는가 (사람의 요소)

| 점검 항목 | 평가 |
|--------|------|
| 정직성 (약속 vs 이행) | |
| 자본배분 능력 (자사주매입/배당/M&A 기록) | |
| 주주 이익 지향성 (지분 보유, 보수) | |
| 오너 마인드 (창업자 vs 전문경영인) | |
| 기업지배구조 (특수관계자 거래, 영업권, 감사) | |
| CEO가 떠나도 정상적으로 운영되는가? | |

**평가 기준** (★1-5):
- ★★★★★: 창업자가 지휘하며, 자본배분이 탁월하고, 이해관계가 완전히 일치
- ★★★★☆: 경영진이 우수하나 사소한 흠이 있음
- ★★★☆☆: 경영진은 적격이나 지배구조 우려 있음
- ★★☆☆☆: 정직성 또는 지배구조 문제 존재
- ★☆☆☆☆: 심각한 정직성 문제 (→하드 부결)

|------|------|---------|------|
| PE (TTM) | | | |

추가 검증 (**반드시 도구로 정밀 계산, 암산 금지**):

- 세 가지 시나리오별 밸류에이션 구간 (도구 출력 결과 채택)

- ★★★☆☆: 적정 밸류에이션, 안전마진 보통
- ★★☆☆☆: 다소 비쌈, 안전마진 부족
- ★☆☆☆☆: 심각한 고평가

#### 6관문: 비중과 판단 규율 (감정적 통제 실패 방지)

다음 감정적 신호를 점검한다:
- FOMO 때문에 사고 싶은가?
- 남이 추천해서 사고 싶은가?
- 5년간 거래정지되어도 받아들일 수 있는가?
- 매수 논거를 200자 이내로 쓸 수 있는가?

### 4단계: 거울 테스트

각 기업에 대해 거울 테스트 문장을 작성한다:

> "나는 ___원에 ___회사를 매수한다, 왜냐하면:
> 1. 이 사업의 본질은 ___이고, 나는 이를 이해하고 있다;
> 2. 이 회사의 해자는 ___이며, 넓어지는 중/좁아지는 중이다;
> 3. 경영진은 ___하며, 신뢰할 만하다/신뢰할 만하지 않다;
> 4. 현재 가격은 내재가치의 ___할에 해당하며, 충분한/불충분한 안전마진이 있다;
> 5. 설령 내가 틀렸더라도, 하방 리스크는 통제 가능/통제 불가능하다, 왜냐하면 ___이기 때문이다."

**다섯 문장을 완전하게 채우지 못하면 = 매수하지 않는다.** "통과" 또는 "미통과"를 명확히 표기한다.

---

### 5단계: 즉시 부결 목록

각 기업에 대해 항목별로 점검하고, 하나라도 해당하면 바로 "부결"로 표기한다:

- [ ] 이 회사가 어떻게 돈을 버는지 명확히 설명할 수 없음
- [ ] 3년 연속 잉여현금흐름이 음(-)이며 개선 조짐이 보이지 않음
- [ ] 경영진에게 정직성 문제 이력이 있음
- [ ] 경쟁우위가 되돌릴 수 없게 잠식되는 중
- [ ] "다음 매수자가 더 높은 가격을 부를 것"에 기대야 수익이 나는 구조 (더 큰 바보 이론)
- [ ] 이 투자가 전액 손실될 경우를 감당할 수 없음
- [ ] 매수 이유가 주로 "남들도 사고 있어서" 또는 "최근 많이 올라서"임
- [ ] 매수 이유를 200자 이내로 명확히 설명할 수 없음

### 6단계: 종합 비교표 출력 (다수 기업일 경우 필수)

여러 기업을 분석할 경우, 반드시 종합 비교표를 생성해야 한다:

| 기업 | 체크리스트 통과? | 능력범위 | 좋은 사업 | 해자 | 경영진 | 안전마진 | 핵심 결론 |
|------|----------------|--------|--------|--------|--------|---------|---------|
| | | ★☆☆☆☆ | ★☆☆☆☆ | ★☆☆☆☆ | ★☆☆☆☆ | ★☆☆☆☆ | |

---

### 7단계: 최종 결론 및 파일 작성

각 기업에 대해 명확한 결론을 내린다(회피하지 않는다):
- ✅ **체크리스트 통과** (X/6관문) — 심층 리서치 단계로 진행 가능
- ❌ **체크리스트 미통과** — 어떤 레드라인이 발동했는지 설명
- ❓ **회색 지대** — 핵심 쟁점이 무엇인지 설명, 투자자가 스스로 판단해야 할 부분을 명시
- N/A — 비상장/매수 불가

완성된 보고서를 `~/버핏체크리스트-[기업명 또는 "다수기업비교"].md`에 작성한다

## 출력 형식 요건

1. 기업마다 독립된 장(章)으로 구성하며, 6관문 평가표 + 핵심 데이터표 + 주요 리스크(3-5개) + 거울 테스트 + 명확한 결론을 포함한다
2. 다수 기업일 경우 마지막에 종합 비교표를 첨부한다
3. 모든 평가는 반드시 ★기호(★1-5)를 사용하며, 반 개 별은 없다
4. 데이터는 반드시 출처 시점을 표기하며, 추정치는 "추정"이라고 명시한다
5. 문말에 결어를 첨부하고, 버핏의 명언("투자의 제1원칙은 손실을 보지 않는 것이다")과 조응시킨다
6. 언어 스타일: 직설적, 날카롭게, 불필요한 말 없이. 버핏/멍거/돤융핑의 어록을 곁들여 코멘트한다

## 핵심 원칙

- **차라리 놓칠지언정 틀린 선택은 하지 않는다**: 체크리스트의 목표는 나쁜 선택을 배제하는 것이지, 최선을 찾는 것이 아니다
- **능력범위를 정직하게 마주한다**: 이해가 안 되면 안 된다고 말하고, 억지로 분석하지 않는다
- **안전마진은 생명선이다**: 좋은 회사도 비싸게 사면 손실이 난다
- **거울 테스트는 건너뛸 수 없다**: 이유를 명확히 설명 못 하면 사지 않는다, 예외는 없다
